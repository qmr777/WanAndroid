# WanAndroid

一个简单的Android版[WanAndroid](https://www.wanandroid.com/)客户端  
实现了主页、体系、导航、问答、登录、收藏等功能，广场和todo未完成，下个版本（如果有的话...）会加上  
大体上还是rxjava3+retrofit2那一套，部分功能复杂的界面用了mvp  
用room和diskLurCache方便的缓存数据
